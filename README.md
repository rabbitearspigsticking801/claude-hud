1. Check that the file finished downloading.
2. Make sure you extracted the zip file if needed.
3. Right-click the app and try Run as administrator.

5. Look for any setup file or config file in the folder.

If the status does not appear:

1. Restart Claude Code.
2. Restart claude-hud.
3. Check whether the display is hidden behind another window.
4. Look for a config setting that controls where the HUD appears.

If Windows blocks the file:

1. Right-click the file.
2. Open Properties.
